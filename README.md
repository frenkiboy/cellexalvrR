# cellexalvrR
cellexalvrR is an R package to prep data for use in the cellexalVR environment, and also provides functions to run in-session calculations

# Install using:

install.packages("devtools")\
library(devtools)

install_github("shambam/cellexalvrR")\
library(cellexalvrR)
